To create a professional online presence that gives me an opportunity to showcase my background, skills, projects, education, and career aspirations. This project will also fulfill the final project requirements while serving as a useful portfolio piece for future employers and personal branding.

**Feedback Loops**
Who needs to provide feedback? When and how?

**Description**
A clean and professional two-page website built with semantic HTML and a single external CSS stylesheet. Project will include static web components (index.html, about.html, & style.css) and planning/description files (plan.md, retrospective.md, scope.md, and README.md). Fully Responsive and hosted on GitHub Pages.
